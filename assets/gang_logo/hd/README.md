## Usage possible

- Emblemes de gangs officiels ou adverses.
- Visuels principaux pour base, raid ou profil de gang.
- Source HD pour exports compacts.

## Integration

- Preferer des chemins stables en minuscules, deja respectes par les fichiers.
- Tester chaque logo sur les fonds de l'interface avant usage: certains logos
  peuvent avoir peu de contraste selon le theme.
- Ne pas renommer `team_gen10.png` avant d'avoir confirme son origine ou cree un
  alias de mapping explicite.

## Droits

de redistribution avant publication ou usage public.
