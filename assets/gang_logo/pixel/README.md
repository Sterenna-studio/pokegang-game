# gang_logo/pixel

Logos pixel art carres pour gangs et badges compacts.

## Contenu

| Fichier | Note |
|---|---|
| `logo_01_skull_full_square.png` | Variante skull. |
| `logo_02_flame_full_square.png` | Variante flame. |
| `logo_03_pink_full_square.png` | Variante pink. |
| `logo_04_shield_full_square.png` | Variante shield. |
| `logo_05_lightning_full_square.png` | Variante lightning. |
| `logo_06_magma_full_square.png` | Variante magma. |
| `logo_07_aqua_full_square.png` | Variante aqua. |

## Usage possible

- Badges de gangs dans une liste ou un classement.
- Icones compactes pour les raids, defenses ou bases.
- Alternatives pixel art aux logos HD quand l'interface demande une lecture plus
  nette en petite taille.

## Integration

- Garder le prefixe numerote pour conserver l'ordre source.
- Creer des alias lisibles dans les mappings plutot que renommer les fichiers.
- Verifier le rendu a petite taille, notamment en `32px` et `48px`.
- Eviter de melanger ces fichiers avec `gang_logo/hd`: les contraintes de
  cadrage ne sont pas identiques.
