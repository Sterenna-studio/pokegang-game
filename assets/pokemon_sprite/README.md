# pokemon_sprite

Sprites Pokemon regroupes par pack ou source.

## Sous-dossiers

| Dossier | Contenu |
|---|---|
| `legendary_fight_by_muzyun/` | Sprites et compositions de combat pour Darkrai, Deoxys et Mewtwo. |
| `poke_pixel_pp_16x16/` | Mini sprites pixel art 16x16 pour Pokemon legendaires ou especes speciales. |
| `pokemon_underwaterby_kyle_dove/` | Pack numerote de sprites underwater et image source du pack. |

## Usage prevu

- Renforcer les rencontres speciales, cinematiques ou combats de boss.
- Fournir des variantes thematiques pour certaines especes.
- Servir de base locale avant publication sur `lab.sterenna.fr/PG` ou mapping dans les helpers de sprites.

## Integration

Avant de brancher un sprite Pokemon dans le runtime:

1. choisir une cle stable par espece ou par variante;
2. ajouter le mapping dans le fichier de donnees ou helper UI concerne;
3. verifier la transparence et le cadrage en combat, PC et collection;
4. documenter ici le nouvel usage.
