01 Introduction

L'API PokéGang permet à des sites et applications tiers d'accéder aux données publiques des gangs. Les joueurs peuvent opt-in depuis leur onglet Compte dans le jeu pour rendre leur fiche visible.

L'API est hébergée sur Supabase Edge Functions (Deno runtime). Toutes les réponses sont en UTF-8 JSON. Les images Pokémon proviennent de Showdown et peuvent être utilisées librement dans vos intégrations.

  • Seuls les joueurs ayant activé public_profile sont accessibles
  • Chaque profil public reçoit un token unique et stable
  • Les données sont synchronisées à chaque sauvegarde cloud (~30s)
ℹ Pour activer votre profil public : onglet Compte → Profil Public / API → cocher la case. Un token unique est généré automatiquement.

02 Endpoints

GET /pokegang-api/gang?name=… Fiche complète d'un gang ▾

Retourne la fiche complète d'un gang public. Recherche par nom de gang (insensible à la casse) ou par token public.

Paramètres

ParamètreTypeRequisDescription
name string optionnel* Nom du gang (insensible à la casse). Ex: Team BZH
token string optionnel* Token public unique du gang. Ex: team-bzh-a1b2c3
* L'un des deux est requis

04 Codes d'erreur

Code HTTPMessageCause
400 Paramètre requis Ni name ni token fourni
404 Gang introuvable Gang inexistant ou profil non public
405 Méthode non autorisée Seule la méthode GET est acceptée
500 Erreur base de données Erreur interne Supabase — réessayer dans quelques secondes

Toutes les erreurs ont la forme : { "ok": false, "error": "message" }

05 Exemples de code

JAVASCRIPT — fetch API
// Fiche d'un gang par nom const BASE = 'https://{project}.supabase.co/functions/v1/pokegang-api'; async function getGang(name) { const res = await fetch(`${BASE}/gang?name=${encodeURIComponent(name)}`); if (!res.ok) throw new Error(`HTTP ${res.status}`); const { data } = await res.json(); return data; } // Par token public (plus stable que le nom) async function getGangByToken(token) { const res = await fetch(`${BASE}/gang?token=${token}`); return (await res.json()).data; } // Utilisation const gang = await getGang('Team BZH'); console.log(gang.profile.reputation); // 54240 console.log(gang.stats.total_caught); // 12388 console.log(gang.showcase[0].sprite); // URL sprite Showdown // Classement top 10 par captures const lb = await (await fetch(`${BASE}/leaderboard?limit=10&sort=caught`)).json(); lb.data.entries.forEach(e => console.log(`#${e.rank} ${e.gang_name}`));
PYTHON 3 — requests
import requests BASE = "https://{project}.supabase.co/functions/v1/pokegang-api" # Fiche par nom def get_gang(name: str) -> dict: resp = requests.get(f"{BASE}/gang", params={"name": name}) resp.raise_for_status() return resp.json()["data"] # Fiche par token def get_gang_by_token(token: str) -> dict: resp = requests.get(f"{BASE}/gang", params={"token": token}) resp.raise_for_status() return resp.json()["data"] # Exemple gang = get_gang("Team BZH") print(gang["profile"]["reputation"]) # 54240 print(gang["stats"]["total_caught"]) # 12388 # Classement lb = requests.get( f"{BASE}/leaderboard", params={"limit": 10, "sort": "caught"} ).json()["data"] for e in lb["entries"]: print(f"#{e['rank']} {e['gang_name']} — {e['reputation']} rep")

06 Intégration Widget

Copiez ce snippet pour afficher une fiche gang sur n'importe quel site. Remplacez TOKEN par le token du gang à afficher. Aucune dépendance externe.

Team BZH
Boss : Dark Muten
⭐ 54 240 réputation
pokegang.sterenna.fr
HTML — widget auto-intégré
<!-- Widget PokéGang — remplacer TOKEN --> <div id="pg-gang-card"></div> <script> (async () => { const TOKEN = 'team-bzh-a1b2c3'; // ← votre token const BASE = 'https://{project}.supabase.co/functions/v1/pokegang-api'; const SHOWDOWN = 'https://play.pokemonshowdown.com/sprites'; // esc() échappe le HTML — appliquez-le à TOUTE donnée distante avant de l'insérer dans le DOM const esc = (v) => String(v ?? '').replace(/[&<>"']/g, c => (({ '&': '&amp;', '<': '&lt;', '>': '&gt;', '"': '&quot;', "'": '&#39;' })[c])); const { data } = await (await fetch(`${BASE}/gang?token=${TOKEN}`)).json(); const { profile, stats, showcase } = data; document.getElementById('pg-gang-card').innerHTML = ` <div style="font-family:'Courier New',monospace;background:#0f0f0f; border:1px solid #2a2a2a;border-left:3px solid #cc3333; border-radius:6px;padding:16px 20px;max-width:360px"> <div style="display:flex;align-items:center;gap:12px;margin-bottom:12px"> <img src="${SHOWDOWN}/trainers/${esc(profile.boss_sprite || 'acetrainer')}.png" style="width:52px;height:52px;image-rendering:pixelated"> <div> <div style="font-size:14px;color:#cc3333;font-weight:700">${esc(profile.gang_name)}</div> <div style="font-size:11px;color:#aaa">Boss : ${esc(profile.boss_name)}</div> <div style="font-size:10px;color:#ffcc5a">⭐ ${profile.reputation.toLocaleString()} rép.</div> </div> </div> <div style="display:flex;gap:6px;flex-wrap:wrap"> ${showcase.filter(s => !s.empty).map(s => `<img src="${esc(s.sprite)}" title="${esc(s.species_fr)} Lv.${esc(s.level)}" style="width:36px;height:36px;image-rendering:pixelated">` ).join('')} </div> <div style="margin-top:8px;font-size:9px;color:#555;text-align:right"> pokegang.sterenna.fr </div> </div>`; })(); </script>
🔒 Sécurité : gang_name, boss_name, species_fr et title sont des saisies joueur. Ne les injectez jamais bruts dans innerHTML — passez-les par esc() (ci-dessus), ou construisez les nœuds avec document.createElement() + textContent. Sinon vous exposez vos visiteurs à du XSS stocké si un champ contient du HTML.
💡 Préférez toujours le token au nom — il est stable même si le joueur renomme son gang. Récupérez-le depuis l'onglet Compte → Profil Public du jeu.
